# Paper Map Viewer

This is a static WebGL viewer for UMAP coordinate tables generated by the Dutch papers pipeline.

Expected CSV columns:

```text
work_id,x,y,publication_year,title
```

The map colors points by `publication_year`.

The full metadata can keep `abstract` for region-labeling workflows, but browser-facing map CSVs should stay small enough to load quickly.

## Export Data

Generate Dutch-paper inputs with:

```text
get_dutch_papers.sql
```

Then run the embedding and UMAP pipeline for those papers. Export a viewer CSV with:

```text
work_id,x,y,publication_year,title
```

Download or combine the exported CSV shards into:

```text
viewer/map_points.csv
```

You can also load a CSV through the file picker in the viewer without renaming it.


## Run Locally

From the repository root:

```powershell
python -m http.server 8000
```

Open:

```text
http://localhost:8000/viewer/
```

The viewer supports zoom/pan, title or work-ID search, zoom-to-result, and click-to-inspect paper details.
